1、if/else 标签

{% if condition1 %} condition1 = True {% elif condition2 %} condition2 = True {% else %} condition1 condition2 = False {% endif %}
{% if condition1 and condition2 %} condition1 and condition2 {% endif %}

2、for 标签

正向迭代

    {% for item in items %}
  • {{ item }}
    • {% endfor %}

反向迭代

    {% for item in items reversed %}
  1. {{ item }}
    2. {% endfor %}

3、ifequal/ifnotequal 标签

{% ifequal one two %} {{ one }} equal {{ two }} {% else %} {{ one }} not equal {{ two }} {% endifequal %} {# 这是注释，不解释#}

4、过滤器

{{ name|upper }} {# 管道字符 #}
{{ name|lower|first|upper }}
{{ address|truncatewords:"3" }}
{{ currenttime}}
{{ currenttime|date:"F j, Y" }}
{{ currenttime|length }}
{{ currenttime|addslashes }} {# 添加反斜杠到任何反斜杠、单引号或者双引号前面 #}

5、include 标签

{% include "footer.html" %}
